# Bihar Land Survey App - Installation Guide

## How to Install APK Received via WhatsApp

### Step 1: Enable Unknown Sources

Before installing, you need to allow installation from unknown sources:

1. Open **Settings** on your phone
2. Go to **Security** or **Privacy**
3. Find **Install unknown apps** or **Unknown sources**
4. Select **WhatsApp** and enable **Allow from this source**

> **Note:** On newer Android versions (8.0+), you'll be prompted automatically when trying to install.

---

### Step 2: Download APK from WhatsApp

1. Open **WhatsApp**
2. Go to the chat containing the APK file
3. Tap on the file **bihar_land_app.apk**
4. Tap **Download** (↓ arrow icon)
5. Wait for download to complete

---

### Step 3: Install the App

1. After download, tap **Open** or **Install**
2. If prompted, tap **Settings** → Enable **Allow from this source**
3. Tap **Install**
4. Wait for installation to complete
5. Tap **Open** to launch the app

---

### Step 4: Allow Permissions

When you first open the app, allow these permissions:
- **Storage** - For viewing documents
- **Internet** - For fetching data

---

## Troubleshooting

### "App not installed" Error
- Free up storage space (need at least 100MB free)
- Uninstall any previous version of the app first
- Re-download the APK file

### "Parse Error"
- APK file may be corrupted
- Request a fresh APK file
- Ensure your Android version is 6.0 or higher

### "Blocked by Play Protect"
1. Tap **More details**
2. Tap **Install anyway**
